## ✨ 特性

- 📦 支持 Pypi 包安装
- 💻 支持多个平台部署
- ⚙️ 支持多个平台签到
- 📢 支持多个平台通知
- ♾️ 支持多个账号签到
- 🕙 支持定时任务设置
- 🆙 支持项目自动更新
